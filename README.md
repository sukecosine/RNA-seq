# Sequencing Fundamentals & QC — Study Notes

Study notes on high-throughput sequencing basics and read quality control, written while
learning bioinformatics analysis. The two notes follow the natural order of a sequencing
run: first how the reads are produced, then how to check that they are good.

1. **Illumina 测序的工作流程** — how an Illumina sequencing library is built and read
   (the workflow that generates the raw reads).
2. **教你读懂 FastQC 质控结果图** — how to read and interpret the FastQC quality-control
   plots for those reads.

## Note

This is a notes / reference repository, not an analysis pipeline. For end-to-end analysis
projects, see my pinned repositories.
